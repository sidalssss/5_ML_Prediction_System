# Machine Learning Prediction System

Bu proje, farklı makine öğrenmesi algoritmalarını (KNN, Logistic Regression, Random Forest) karşılaştırarak veri seti üzerinde tahminler yapan bir analiz sistemidir.

## Özellikler
- **Çoklu Model Desteği:** KNN, Lojistik Regresyon ve Random Forest modellerinin eğitimi.
- **Veri Analizi:** Pandas ve NumPy ile veri ön işleme süreçleri.
- **Görselleştirme:** Matplotlib ile model performanslarının karşılaştırılması.

## Kullanılan Kütüphaneler
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Çalıştırma
```bash
python prediction_system.py
```
